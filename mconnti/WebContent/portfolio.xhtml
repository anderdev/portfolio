Organizer, financial helper.

I had first developed this application using Delphi 7 when I was living in Australia, because excel wasn't enough to maintain my expenses. Then during my Java course I changed it to use Struts 1.2, Hibernate and Servlets and during the years I've been changing it to use frameworks that I want to learn, SEAM, SPRING, JSF.....

Currently this application is deployed using Spring 2.5, Hibernate JSF 1.2 with richfaces, but I'm already working on an upgrade to Spring 3 and JSF2 with Primefaces.

I use this application as my stage area.

Its code is on my public repository: https://github.com/anderdev/portfolio

LibraryJ, onLine library.

Library onLine is a web application developed using JSF2 with Primefaces and Jersey.
I developed this application to learn how to have an web application consuming data from a server application.

I have developed the ServerApplication as well, that uses Spring3, Hibernate and Jersey.
This application consumes json's from web application and returns json to the web application.

Its code is on my public repository: https://github.com/anderdev/portfolio
